git clone command used to clone repository from git to computer<br>
cd command used to change directory<br>
ls command used to list all files in folder<br>
ls -a command used to access hidden files that are accessible in terminal only<br>
git status command used to check status of code
mkdir "Foldername" to create new directory or folder
git init command used to make file or dirctory as a part of git
git remote add origin command used to add new repository as git repositories are remote
git branch command used to check on which branch we are working
git branch -M "newname" command used to rename branch
git checkout -b "newbranchname" command used to create a new branch
git checkout -d "branchname" command used to delete branch
git checkout "branchname" command used to move to other branch
git pull origin main comand used to pull code from github to your local repo
git log used to check commit histories
Staged Changes
git reset "filename" to reset specific file
git reset to reset all files
Commited Changes
git reset HEAD~1 command used to reset recently commited change as head represent last commited change         (for single commit)
git reset "hash no" command used for multiple commit reseting and hash represent that commit until that we want to reset
get reset --hard command can reset changes or commits from github and vscode as well
fork is rough copy of repo as it involves creating another repo by copying old one
